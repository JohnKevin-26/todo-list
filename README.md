# 📝 To-Do List Pro

Aplicação web de lista de tarefas desenvolvida com HTML, CSS e JavaScript puro.

---

## 🚀 Demonstração

![Preview do projeto](assets/screenshot.png)

---

## ⚙️ Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas da lista
- Filtros dinâmicos (Todas, Pendentes, Concluídas)
- Contador automático de tarefas
- Salvamento automático com LocalStorage

---

## 🧠 Conceitos aplicados

- Manipulação do DOM
- Eventos em JavaScript
- Funções e organização de código
- Persistência de dados no navegador (LocalStorage)
- Estruturação de interface com HTML e CSS

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com foco em prática de lógica de programação e evolução no desenvolvimento Front-End, simulando uma aplicação real de produtividade.

---

## 📌 Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
